# Development Setup

Get TimeTiles up and running for development.

## Prerequisites

- Node.js 24+
- pnpm
- Docker and Docker Compose

## Quick Start

```bash
# Clone the repository
git clone <repository-url>
cd timetiles

# Install dependencies
pnpm install

# Start PostgreSQL database
make up

# Run database migrations
cd apps/web && pnpm payload:migrate

# Start development server
pnpm dev
```

Open http://localhost:3000 to see the application.

## Creating Your First Admin User

```bash
cd apps/web
pnpm payload create:user
```

Follow the prompts to create an admin user, then log in at http://localhost:3000/admin

## Development Commands

From the project root:

- `pnpm dev` - Start development server
- `pnpm build` - Build all packages
- `pnpm test` - Run tests
- `pnpm lint` - Run linting
- `pnpm format` - Format code with Prettier

From `apps/web`:

- `pnpm dev` - Start Next.js dev server
- `pnpm payload:generate` - Generate TypeScript types
- `pnpm seed` - Seed development data
- `pnpm test` - Run Vitest tests

## Docker Commands

- `make up` - Start PostgreSQL
- `make down` - Stop services  
- `make logs` - View logs

## Troubleshooting

### Port 3000 in use
```bash
lsof -ti:3000 | xargs kill -9
```

### Database connection issues
```bash
make down && make up
```

### Reset everything
```bash
make clean && make up
```

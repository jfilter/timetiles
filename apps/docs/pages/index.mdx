# TimeTiles Documentation

> **Note:** TimeTiles is under active development. Features, APIs, and documentation may change frequently as we continue to improve the platform.

Welcome to the comprehensive documentation for TimeTiles - a powerful spatial data management platform built with Next.js, Payload CMS, and PostgreSQL with PostGIS.

## What is TimeTiles?

TimeTiles is a monorepo application designed for spatial data management and content administration. It provides:

- **Spatial Data Management**: PostgreSQL with PostGIS for advanced spatial operations
- **Content Management**: Payload CMS for structured content and user management
- **Modern Stack**: Next.js 15, React 19, TypeScript, and Tailwind CSS
- **Developer Experience**: Turborepo monorepo with pnpm and comprehensive tooling

## Quick Start

Get up and running with TimeTiles in minutes:

```bash
# Clone and setup
git clone <repository-url>
cd timetiles
make setup

# Start development
make dev

# Set up Payload CMS
cd apps/web
pnpm payload create:user
```

## Import Pipeline Detail

The 6-task sheet processing pipeline with file readers, transforms, and review loops. For stage-by-stage documentation, see [Processing Stages](./data-processing-pipeline/stages).

```mermaid
flowchart TD
    subgraph ENTRY["Entry Points"]
        E1["File Upload<br/>CSV / Excel / ODS / JSON"]
        E2["URL Fetch<br/>HTTPS + auth config"]
        E3["Scraper Output<br/>CSV from Podman"]
    end

    subgraph FILE_PROC["File Processing"]
        READER["streamBatchesFromFile<br/>Async generator"]
        PAPA["Papa Parse<br/>CSV streaming with backpressure<br/>~3MB memory per batch"]
        EXCEL["xlsx (SheetJS)<br/>Sheet → CSV sidecar conversion"]
        JSON_CSV["convertJsonToCsv<br/>JSON API → flat CSV"]
    end

    subgraph TRANSFORMS["Data Transforms (applied at each stage)"]
        TR_RENAME["rename — Move field path"]
        TR_DATE["date-parse — Parse date strings"]
        TR_STR["string-op — upper / lower / replace / expression"]
        TR_CONCAT["concatenate — Join fields"]
        TR_SPLIT["split — Split field by delimiter"]
    end

    subgraph DEDUP["Stage 1: Analyze Duplicates"]
        GEN_ID["generateUniqueId<br/>per dataset ID strategy"]
        INT_DUP["Internal duplicates<br/>Same unique ID within file"]
        EXT_DUP["External duplicates<br/>Query existing events by ID"]
        SKIP_LIST["Build skip-row list<br/>for later stages"]
    end

    subgraph SCHEMA_DET["Stage 2: Detect Schema"]
        PROG_BUILDER["ProgressiveSchemaBuilder<br/>Incremental type inference"]
        GEO_DETECT["detectFlatFieldMappings<br/>Find geocoding-capable fields"]
        FIELD_STATS["Field statistics<br/>Type distribution, nullability"]
    end

    subgraph SCHEMA_VAL["Stage 3: Validate Schema"]
        COMPARE["Compare with latest version<br/>Breaking vs non-breaking changes"]
        QUOTA_CHECK["Check event quota<br/>Would import exceed limits?"]
        REVIEW_GATE{"Needs<br/>Review?"}
    end

    subgraph SCHEMA_VER["Stage 4: Create Schema Version"]
        NEW_VER["Create dataset-schemas record<br/>Increment version number"]
    end

    subgraph GEOCODE["Stage 5: Geocode Batch"]
        EXTRACT_LOC["Extract unique locations<br/>normalizeGeocodingAddress"]
        CACHE_CHECK["Check LocationCache<br/>Skip already-geocoded"]
        PROVIDERS["Multi-provider geocoding<br/>Nominatim / Google / OpenCage"]
        RESULT_MAP["Store results map<br/>address → lat, lng, confidence"]
    end

    subgraph CREATE["Stage 6: Create Events"]
        BUILD_EVENT["createEventData<br/>Transform row → event object"]
        LOOKUP_GEO["Look up geocoding results<br/>from results map"]
        DEDUP_ID["Generate deduplication ID"]
        DENORM["Denormalize access fields<br/>isPublic, catalogOwnerId"]
        BULK_INSERT["bulkInsertEvents<br/>INSERT ... ON CONFLICT DO NOTHING"]
    end

    subgraph OUTCOME["Outcomes"]
        COMPLETED["COMPLETED<br/>Events in database"]
        NEEDS_REVIEW["NEEDS_REVIEW<br/>Awaiting user approval"]
        FAILED["FAILED<br/>Error logged to ingest-jobs"]
    end

    E1 --> READER
    E2 --> READER
    E3 --> READER
    E1 -.->|"JSON API"| JSON_CSV --> READER
    READER -->|".csv"| PAPA
    READER -->|".xlsx / .ods"| EXCEL --> PAPA

    PAPA --> TRANSFORMS
    TRANSFORMS --> DEDUP

    DEDUP --> GEN_ID --> INT_DUP --> EXT_DUP --> SKIP_LIST

    SKIP_LIST --> SCHEMA_DET
    SCHEMA_DET --> PROG_BUILDER --> GEO_DETECT --> FIELD_STATS

    FIELD_STATS --> SCHEMA_VAL
    SCHEMA_VAL --> COMPARE --> QUOTA_CHECK --> REVIEW_GATE
    REVIEW_GATE -->|"Yes"| NEEDS_REVIEW
    REVIEW_GATE -->|"No"| SCHEMA_VER
    NEEDS_REVIEW -->|"User approves"| SCHEMA_VER

    SCHEMA_VER --> NEW_VER

    NEW_VER --> GEOCODE
    GEOCODE --> EXTRACT_LOC --> CACHE_CHECK --> PROVIDERS --> RESULT_MAP

    RESULT_MAP --> CREATE
    CREATE --> BUILD_EVENT --> LOOKUP_GEO --> DEDUP_ID --> DENORM --> BULK_INSERT

    BULK_INSERT --> COMPLETED
    DEDUP -.->|"Error"| FAILED
    SCHEMA_DET -.->|"Error"| FAILED
    GEOCODE -.->|"Error"| FAILED
    CREATE -.->|"Error"| FAILED
```

## Data Organization & Access Control

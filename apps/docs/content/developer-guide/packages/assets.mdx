# Assets Package

Shared logos and visual assets for the TimeTiles monorepo. All assets are automatically available after running `make setup`.

## Available Assets

### Static SVG Logos

- `logo-static.svg` - Base logo
- `logo-with-text-horizontal.svg` / `logo-with-text-horizontal-tight.svg` - Horizontal layouts
- `logo-with-text-vertical.svg` / `logo-with-text-vertical-tight.svg` - Vertical layouts

### Animated SVG Logos

- `logo-1.svg` through `logo-10.svg` - 10 animation variants (fade-ins, rotations, pulses, waves)

### Raster Exports (PNG/JPEG)

Available in 256px, 512px, 1024px, 2048px, and 4096px resolutions. Stored via Git LFS.

## Usage

Import assets directly in your code:

```typescript
import logo from "@timetiles/assets/logos/static/logo-static.svg";
import logoWithText from "@timetiles/assets/logos/static/logo-with-text-horizontal.svg";
import favicon from "@timetiles/assets/logos/static/logo-256.png";

// Use in React components
<img src={logo} alt="TimeTiles" />
```

Next.js apps must include asset handling in their webpack config (already configured in `apps/web` and `apps/docs`).

## Notes

- PNG/JPEG files are stored via Git LFS (automatically handled by `make setup`)
- Don't modify logo colors or gradients
- Preserve aspect ratios when scaling
- Use horizontal variants for headers, vertical for sidebars

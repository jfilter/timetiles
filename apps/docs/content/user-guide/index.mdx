# User Guide

Documentation for using TimeTiles.

## Overview

This guide covers how to use TimeTiles to ingest, explore, and share temporal geospatial data on interactive maps.

## Documentation Structure

### [Getting Started](./getting-started)

Learn how to ingest data, navigate the interface, explore events, and use the timeline and map features.

### [Data Ingestion](./data-ingestion)

How the ingestion pipeline works — supported formats, schema detection, geocoding, scheduling, and troubleshooting.

### [Use Cases](./use-cases)

Real-world examples of how to use TimeTiles for journalism, research, activism, and data exploration.

## Key Features

- **Ingest**: Upload CSV, Excel, or ODS files — or set up automated URL ingestion on a schedule
- **Explore**: Interactive map with clustering, timeline histogram, and dynamic filters
- **Manage**: Organize data into catalogs and datasets, with schema versioning and deduplication
- **Share**: Every view has a unique URL with filters preserved — bookmark or share specific views
- **Account**: Self-service data export (ZIP) and account deletion with grace period

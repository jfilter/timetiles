TimeTiles Cartographic Logo

Refined earth-tone design following the Cartographic Design System

Design Philosophy

  • Earth-Tone Palette: Navy, terracotta, forest green, and cream replace generic blue gradients
  • Layered Geometry: Rectangular tiles suggest temporal map slices and geospatial data layers
  • Cartographic Details: Subtle coordinate grid, compass rose, and location markers evoke vintage maps
  • Editorial Typography: Playfair Display serif brings authority and sophistication
  • Sharp Precision: 2px border radius maintains cartographic accuracy
  • OKLCH Colors: Perceptual uniformity ensures consistent appearance across displays

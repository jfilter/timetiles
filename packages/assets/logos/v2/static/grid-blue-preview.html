Blue & Beige Grid Variations

Grid-Focused • Cartographic Design System

Design Direction

These iterations explore the grid-like aesthetic using blue shades and beige/gray tones from the Cartographic Design System, eliminating all green colors. Each variation emphasizes different aspects of geospatial mapping and temporal data visualization.

  • Blue Palette Cartographic blue (#6495ed), navy (#4a5568), charcoal (#404040) for data and temporal emphasis
  • Beige/Gray Palette Parchment (#f8f5f0), cream (#e8e4dd), gray tones for neutral base and subtle structure
  • Grid Emphasis Strong horizontal and vertical lines, coordinate systems, and technical precision
  • Temporal Layers Overlapping tiles suggest time slices and historical data depth
  • Cartographic Details Coordinate markers, compass elements, measurement indicators, and blueprint styling
  • No Green Forest green completely removed in favor of blue/gray/beige harmony

Evolution from Original Design

Original Blue Gradient Logo

  • Cyan/sky blue gradients (#0ea5e9 → #0284c7)
  • Abstract mosaic tiles
  • Generic tech aesthetic
  • Scattered circular accents
  • Light grid overlay

New Grid Blue Iterations

  • Cartographic blue + navy + beige (#6495ed, #4a5568)
  • Structured grid-first design
  • Technical/surveyor aesthetic
  • Coordinate systems and markers
  • Bold, prominent grid lines

Recommendations

  • Version 1 - Clean Grid Best for modern, approachable branding. Simple enough to work at small sizes while maintaining visual interest.
  • Version 2 - Temporal Layers Strongest conceptual alignment with "TimeTiles" name. Clearly shows temporal progression and data history.
  • Version 3 - Map Mosaic Most cartographically authentic. Excellent for geospatial/mapping contexts but may be too detailed at small sizes.
  • Version 4 - Blueprint Most distinctive and memorable. Technical precision aesthetic perfect for professional GIS/mapping tools.

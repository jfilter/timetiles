## Design Principles

All variants follow these principles:

1. **Simplicity** - Reduced from v1's 19+ elements to 3-20 elements max
2. **Clarity** - Each element serves a purpose (time, space, or tiles)
3. **Versatility** - Works on light and dark backgrounds
4. **Scalability** - Clean at all sizes (favicon to billboard)
5. **Cartographic Aesthetic** - Inspired by maps, precision, editorial design

## Complexity Levels

- **Minimal** (3-5 elements) - Extremely clean, modern tech aesthetic
- **Clean** (8-12 elements) - Balanced simplicity with clear concept expression
- **Moderate** (15-20 elements) - Visual interest without v1's overwhelming complexity
